# 个人任务管理系统 v1.0.1

一个功能完整的本地个人任务管理网页应用。

## 功能特性

- 📊 **仪表盘** - 日历视图、每日任务、每日一句、数据统计面板
- 📝 **项目管理** - 创建项目、子任务、进度跟踪、拖拽排序
- 🎯 **长期目标** - 1年以上目标规划、阶段管理、拖拽排序
- 💭 **心情记录** - 记录每日心情、历史回顾、心情趋势图
- 💾 **本地存储** - 所有数据保存在浏览器本地
- 📦 **数据备份与恢复** - 支持完整备份、选择性备份、数据恢复
- 🎨 **多主题切换** - 深色/浅色/海洋/森林四种主题
- 🖱️ **拖拽排序** - 自定义项目和目标的显示顺序
- ✨ **操作反馈** - Toast 提示、任务完成动画

## 技术栈

- HTML5 + CSS3 + JavaScript
- Tailwind CSS - 样式框架
- Chart.js - 数据可视化
- Font Awesome - 图标库
- SortableJS - 拖拽排序

## 快速开始

1. **打开项目**
   - 直接在浏览器中打开 `index.html` 文件
   - 或使用本地服务器（推荐）

2. **使用本地服务器（推荐）**
   ```bash
   # 使用 Python
   python -m http.server 8000
   
   # 使用 Node.js (http-server)
   npx http-server
   
   # 使用 VS Code Live Server 扩展
   ```

3. **访问应用**
   - 在浏览器中访问 `http://localhost:8000`

## 使用说明

### 仪表盘
- 查看日历和每日任务
- 添加临时任务
- 阅读每日一句名言
- 查看数据统计（本周完成任务、项目进度、心情趋势图）

### 项目管理
- 创建新项目（设置名称、描述、时间范围）
- 添加子任务（支持每日重复）
- 查看项目进度和完成趋势图表
- 拖拽排序自定义项目显示顺序（拖动左侧手柄）

### 长期目标
- 创建至少1年的长期目标
- 划分阶段和小目标
- 跟踪目标完成进度
- 拖拽排序自定义目标显示顺序（拖动左侧手柄）

### 心情记录
- 选择预设心情或自定义
- 添加备注说明
- 查看历史心情记录
- 查看心情趋势图（仪表盘）

### 用户信息
- 点击头像上传自定义头像
- 点击昵称编辑显示名称

### 数据备份与恢复
- 点击侧边栏「备份数据」可选择备份全部或部分数据
- 点击侧边栏「恢复数据」可导入之前的备份文件
- 备份文件包含时间戳，便于管理多个版本

### 主题切换
- 点击侧边栏「切换主题」可选择：
  - 🌙 深色主题（默认）
  - ☀️ 浅色主题
  - 🌊 海洋主题
  - 🌲 森林主题
- 主题偏好会自动保存

## 数据存储

所有数据存储在浏览器的 LocalStorage 中，包括：
- 用户头像和昵称
- 项目和任务
- 长期目标
- 心情记录
- 每日一句历史

**注意**：清除浏览器数据会删除所有存储的内容。

编辑 `quotes.json` 文件来添加或修改每日一句的内容：

```json
{
  "quotes": [
    "你的名言1——作者",
    "你的名言2——作者"
  ]
}
```

## 开发说明

### 修改样式
编辑 `style.css` 文件来自定义样式。

### 修改功能
各功能模块独立拆分在 `modules/` 目录下，便于维护和扩展。

### 添加新模块
1. 在 `modules/` 目录创建新的 JS 文件
2. 在 `index.html` 中引入
3. 在 `main.js` 中初始化
